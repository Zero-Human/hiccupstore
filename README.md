### 📜 프로젝트를 통해 얻은 경험
- Oauth2를 이용하여 SNS인증로그인을 경험해볼수있었습니다.<br/>
- DB 모델링과 프로젝트 기획 및 설계를 통해서 웹 서비스를 구성할 수 있는지 경험할 수 있었습니다.<br/>
- JQuery를 통해서 프론트와 백의 통신에 대해서 알 수 있었습니다.
- 디버깅를 통해서 버그를 어떻게 수장할 수 있는지 배울 수 있었습니다.

<br/>


### 🏅 프로젝트 내 역할 및 기여도
 - 장비구니 기능 구현
 - 상품,리뷰, 문의게시판 조회 등록 수정 삭제
 - 찜 기능 구현
